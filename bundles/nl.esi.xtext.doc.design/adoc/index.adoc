= XPlus Design

ifdef::output-site[]
ifdef::output-html[]
TIP: The design document is also available in the Eclipse Help (via menu menu:Help[Help Contents] and it can be link:xplus-design.pdf[downloaded as a PDF].
endif::output-html[]
endif::output-site[]

This design document describes the high level design of the XPlus library, a collection of base Xtext languages and language infrastructure components that can be reused across other projects. XPlus provides the foundational language definitions and tooling support that other domain-specific languages and applications can build upon.

CAUTION: Reading this developer guide requires intermediate to advanced knowledge about programming for the Eclipse Modeling Framework (EMF).
A good book to read as reference is https://www.google.com/search?q=ISBN+13%3A+9780321331885[EMF: Eclipse Modeling Framework].
